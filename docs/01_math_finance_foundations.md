## 3. 核心概念

### 收益率（Return）

一句人话：收益率是“今天相对昨天涨跌了多少”。

常用表达：

- 简单收益率（Simple Return）
- 对数收益率（Log Return）

公式：

```text
r_t = P_t / P_{t-1} - 1
g_t = ln(P_t / P_{t-1})
```

符号说明：

- `P_t`：第 `t` 天价格
- `r_t`：第 `t` 天简单收益率
- `g_t`：第 `t` 天对数收益率

### 波动率（Volatility）

一句人话：波动率描述的是“价格动得有多剧烈”。

常见做法是对一段时间内的收益率求标准差（Standard Deviation）。

```text
\sigma_t = std(r_{t-w+1}, ..., r_t)
```

其中：

- `w`：滚动窗口长度（Rolling Window）
- `sigma_t`：第 `t` 天的滚动波动率

### 标准分数（Z-score）

一句人话：Z-score 用来衡量一个变量当前值偏离自己近期平均水平有多远。

```text
z_t = (x_t - \mu_t) / \sigma_t
```

## 4. 黄金为什么不是孤立价格序列

黄金（Gold）通常受以下变量共同影响：

- 美元（U.S. Dollar）
- 实际利率（Real Yield）
- 通胀预期（Inflation Expectations）
- 风险偏好（Risk Sentiment）
- 股票市场压力（Equity Stress）
- 波动率状态（Volatility Regime）

直觉上：

- 美元偏强时，黄金常常承压
- 实际利率上升时，黄金的持有吸引力可能下降
- 风险上升时，黄金有时会被视为避险资产（Safe-Haven Asset）

## 5. 为什么要先研究收益率而不是裸价格

- 价格常常非平稳（Non-stationary）
- 收益率更适合比较不同时期
- 很多统计模型默认处理的对象更接近“变化量”而不是“水平值”

## 6. 常见误区

- 只看价格，不看收益率
- 只看相关性（Correlation），不理解背后的经济机制
- 把黄金当成独立资产，不考虑美元和利率
- 看到模型准确率就默认有交易价值
